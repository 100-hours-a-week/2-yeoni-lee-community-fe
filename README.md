# 🎀 아무 말 대잔치 - Memo Web App 🎀
### 핑크 픽셀 컨셉의 감성적인 메모 웹 서비스
### 📝 "나만의 공간, 자유롭게 아무 말이나 적어보세요!"

💡 프로젝트 소개
**"아무 말 대잔치"**는 사용자가 익명으로 자유롭게 메모를 작성하고, 수정하고, 삭제할 수 있는 웹 게시판 입니다.

## ✨ 주요 기능
### 📌 회원가입 & 로그인
- 이메일 & 닉네임 기반 계정 생성
- 프로필 사진 업로드 가능
- 로그인 후 메모 작성 가능

### 📌 메모 CRUD
- 📝 메모 작성 / ✏️ 수정 / 🗑️ 삭제 기능
- 📷 이미지 첨부 가능
- 🔄 실시간 좋아요 & 댓글 기능

### 📌 사용자 정보 관리
- 프로필 수정 (닉네임 변경 & 프로필 사진 변경)
- 내 게시물 관리 기능

### 📌 보안 & 데이터 처리
- JWT 기반 로그인 세션 유지
- SQL Injection 방지
- XSS 보호를 위한 입력 필터링


## 🎨 UI 컨셉
### 🎀 핑크 & 픽셀 디자인
- 복고풍의 감성을 살린 픽셀 스타일 UI
- 귀여운 UI 요소 & 부드러운 색감 조합

### 🖥️ 반응형 웹 디자인
- 모바일 & PC 모두 지원
- 어떤 기기에서도 메모 작성 가능!
